# 观点判断 / Opinions

这里记录我看到的视频、评论、观点、金句，并做逻辑拆解。

重点不是为了反驳别人，而是训练自己的判断力。

## 记录对象 / What to Capture

- 视频观点
- 评论区言论
- 社交媒体金句
- 日常听到的判断
- 我自己的第一反应

## 拆解方式 / Breakdown

```text
原观点是什么？
我的第一反应是什么？
它的结论是什么？
它的前提是什么？
有没有隐含假设？
有没有反例？
它在什么条件下可能成立？
更严谨的表达是什么？
```

## 目标 / Goal

看到一句话，不只是接受或反驳，而是判断它的结构、边界和论证。
