# 临时收集区 / Inbox

这里放还没整理好的内容。

适合放：

- 突然想到的一句话
- 刷题时卡住的点
- 工程开发中的问题
- 视频或评论区看到的观点
- 没想清楚但值得保留的判断

## 使用原则 / Principles

先收集，不急着分类；先保留，不急着完美。

```text
capture first, organize later
```

以后整理时，可以把这里的内容移动到：

- `01-thinking/`
- `02-algorithm/`
- `03-software-engineering/`
- `04-opinions/`
- `05-articles/`
- `06-reading/`
