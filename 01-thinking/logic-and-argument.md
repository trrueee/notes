# 观点和论证 / Claim and Argument

## 一句话理解 / One-line Summary

观点是你主张什么，论证是你凭什么这么说。  
A claim is what you assert; an argument is how you support it.

## 核心区分 / Core Distinction

- 观点 / Claim：一个判断、结论、主张
- 论证 / Argument：支持这个观点的理由、证据、推理过程

## 常见问题 / Common Problems

- 只有观点，没有论证
- 论证不足以推出结论
- 把相关性说成因果
- 把局部经验说成普遍规律
- 偷换概念
- 忽略边界和反例

## 判断模板 / Thinking Template

看到一句话时，可以问：

```text
1. 它的结论是什么？
2. 它的理由是什么？
3. 这个理由真的能推出结论吗？
4. 有没有隐含前提？
5. 有没有反例？
6. 在什么条件下它可能成立？
7. 更严谨的表达是什么？
```

## 例子 / Example

原观点：

> 认知高的人不抠字眼。

分析：

- 结论：认知高的人不抠字眼
- 隐含前提：抠字眼代表低认知、抓不住重点
- 问题：抠字眼可能是无意义纠缠，也可能是对概念边界敏感
- 反例：法律、哲学、科学、合同、工程接口定义中，概念精确很重要

更严谨的表达：

> 认知成熟的人不会为了赢而故意曲解别人的措辞，但在关键概念上会重视准确性。

## 我的理解 / My Understanding

很多听起来很有道理的话，其实只是情绪、经验或姿态，不一定有完整论证。

成熟的判断不是马上相信或马上反驳，而是拆出它的结构和边界。
