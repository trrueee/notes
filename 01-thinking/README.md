# 思考框架 / Thinking

这里沉淀底层思维方式。

重点主题：

- 观点和论证 / Claim and Argument
- 边界思维 / Boundary Thinking
- 相关性与因果 / Correlation and Causation
- 隐含前提 / Hidden Assumptions
- 反例意识 / Counterexamples
- 如何判断一句话有没有道理

## 整理目标 / Goal

把零散反应变成可复用的判断框架。

## 常用问题 / Useful Questions

```text
1. 这句话的观点是什么？
2. 它的论证是什么？
3. 这个论证真的能推出结论吗？
4. 有没有隐含前提？
5. 有没有反例？
6. 它在什么条件下可能成立？
7. 更严谨的表达是什么？
```
