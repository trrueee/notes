# 标签规则 / Tag Rules

标签建议用英文，方便搜索和统一。

## 常用标签 / Common Tags

```text
#algorithm
#thinking
#logic
#engineering
#boundary
#reflection
#opinion
#writing
#java
#leetcode
#debugging
#architecture
#testing
#delivery
#reading
```

## 使用原则 / Principles

- 标签不要太多，每篇 2-5 个即可
- 优先使用已有标签
- 标签表示主题，不表示情绪
- 对长期主题使用稳定标签，例如 `#boundary`、`#reflection`

## 示例 / Example

```md
## Tags
#algorithm #prefix-sum #hashmap #reflection
```
