# 写作规则 / Writing Rules

## 基本原则 / Basic Principles

- 中文为主，关键标题和概念附英文
- 不追求一次写完美，先记录，再整理
- 保留第一反应，因为它能反映真实思考路径
- 不只写结论，也写推理过程
- 不把经验绝对化，要写清边界和反例

## 推荐结构 / Suggested Structure

大多数笔记可以尝试包含：

```text
背景 / Background
第一反应 / First Reaction
分析 / Analysis
边界 / Boundaries
例子 / Examples
反例 / Counterexamples
结论 / Conclusion
复盘 / Reflection
```

## 中英双语策略 / Bilingual Strategy

不用每句话都中英双写。

推荐：

```text
标题：中文 + 英文
小标题：中文 + 英文
核心概念：中文 + 英文
正文：中文为主
关键总结：可以补一句英文
```

例子：

```md
# 边界思维 / Boundary Thinking

## 一句话理解 / One-line Summary

边界决定判断是否成立。  
Boundaries decide whether a judgment holds.
```
