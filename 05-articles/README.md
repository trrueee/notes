# 文章 / Articles

这里放从笔记中发展出来的长文。

不是所有笔记都需要变成文章。只有当一个主题反复出现，并且已经有较完整的判断框架时，再整理成文章。

## 目录 / Structure

```text
drafts/      草稿
published/   完成稿
```

## 文章来源 / Sources

文章可以来自：

- 多篇 thinking 笔记
- 多道 algorithm 复盘
- 多个 engineering 问题
- 多个 opinions 判断
- 阅读和实践中的长期观察

## 写作目标 / Goal

不是为了堆字数，而是把一个问题讲清楚。
